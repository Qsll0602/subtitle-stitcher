# 字幕拼接工具 - Subtitle Stitcher

## 简介

一个基于Web浏览器的字幕图片拼接工具，专门用于处理电影、视频中的字幕截图。用户可以通过多种方式上传多张图片，可视化调整裁剪区域，并将选定的部分智能拼接成一张新图片。所有操作均在本地浏览器中完成，无需上传服务器。

本项目的源代码基本都由AI生成，目前一共迭代了三版，动机是作者在聊天软件中分享一些视频或者电影的内容时，相比于发送视频和单张图片，觉得多张字幕拼接的方式在传递信息上更为高效，但一直找不到合适的轻量级字幕拼接工具，因此催动AI写了这样一个本地的拼接器，目前来看凑合能用；仅供学习参考（如果值得的话）。

## 功能

### 上传方式
- **文件选择**：点击"选择图片"按钮上传
- **拖拽上传**：直接将图片文件拖入工作区（比如从微信聊天框中直接拖到网页的工作区中，作者就很喜欢这样用）
- **粘贴截图**：按Ctrl+V粘贴剪贴板中的截图

### 裁剪
- **可视化调整**：通过拖动绿色裁剪边缘选择区域，相同裁剪方向上，裁剪的另外一个尺寸固定为图片在这个方向上的尺寸
- **双向**：支持横向和纵向裁剪模式
- **独立设置**：每张图片可单独设置裁剪区域和方向（但是方向单独设置似乎并没有什么实际意义）

### 编辑
- **拖拽排序**：通过拖拽调整图片顺序
- **快速删除**：点击右上角删除按钮移除单张图片
- **批量清除**：一键清除所有图片重新开始
- **结果再利用**：将生成的图片拖回工作区继续编辑

### 拼接
- **自动对齐**：以第一张图片方向为基准自动拼接
- **输出**：保持原始图片质量进行拼接
- **预览**：处理完成后右侧实时显示结果

### 导出
- **一键下载**：下载拼接后的结果图片
- **格式支持**：输出为PNG格式图片


### 详细操作
1. **图片上传**
   - 点击"选择图片"按钮选择图片文件
   - 或直接将图片文件拖入左侧工作区
   - 或复制截图后在工作区按Ctrl+V粘贴

2. **裁剪设置**
   - 点击图片选中（蓝色边框表示选中）
   - 拖动绿色裁剪框的边缘调整大小
   - 拖动裁剪框内部调整位置
   - 点击"切换为横向/纵向"改变裁剪方向

3. **顺序调整**
   - 拖动图片周围的灰色区域调整顺序
   - 序号会实时更新

4. **处理图片**
   - 调整完成后点击"处理图片"按钮
   - 右侧将显示拼接后的效果

5. **导出结果**
   - 点击"下载结果"保存图片到本地
   - 如需继续编辑，可将结果图片拖回左侧工作区

### 运行环境
- Web浏览器
- 本地文件访问权限（用于文件上传）
- 无需额外安装或配置


### 运行方法
1. 双击`index.html`文件
2. 或在浏览器中打开`index.html`



## 注意事项

1. **安全**：所有图片处理均在本地浏览器完成，不会上传到任何服务器
2. **建议**：建议单次处理不超过15张图片，单张图片不超过5MB
3. **格式支持**：支持常见图片格式（JPG、PNG、GIF等）
4. **操作提示**：拖拽排序时请拖动图片周围的灰色区域，而非图片本身
5. **浏览器兼容**：建议使用较新版本浏览器
